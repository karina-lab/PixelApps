# Pixel Apps

A Magisk module that brings exclusive applications and features from the **Pixel 10 series** to your device. 

> [!WARNING]  
> **Disclaimer:** This module is made mostly for fun and educational purposes. **Most features will not work.** Please do not submit issues regarding broken functionality. Only open issues if you have suggestions for adding new apps and features to the module.

## Includes

* **Pixel Sysconfigs**
* **Pixel 10 Wallpapers**
* **Pixel Studio**
* **Pixel Screenshots**
* **Device Intelligence**
* **Gemini** as a system application.
* **Password Manager** as a system application.

## Why

Honestly? Mostly for flexing and seeing what the new Pixel software looks like. Don't expect a daily-driver experience here.

| App / Component | Current Status / Benefit |
| :--- | :--- |
| **Pixel Sysconfigs** | Lets you update some of these apps via Google Play. Also allows downloading Pixel Journal and other stuff, just like you would from APKMirror. |
| **Pixel Wallpapers** | Ability to select them directly from the Pixel launcher. You could just download them from the Internet, you know? |
| **Pixel Studio** | Next to nothing. Likely blocked by the unlocked bootloader anyway. |
| **Pixel Screenshots** | AI features are dead, but it’s still a decent-looking viewer. |
| **Device Intelligence** | Gives you a **Magic Cue** entry in Settings just for the aesthetic. |
| **Gemini** | Maybe it will offer deeper integration in future updates? |
| **Password Manager** | System-level shortcut. Possibly useful in the future. |

## Requirements

* **Root:** Magisk, KernelSU, or APatch.
* **Target:** Designed for Pixel devices, but compatible with most modern AOSP-based ROMs.
